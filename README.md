# ✧ Mystical Tarot - 신비로운 타로의 세계 ✧

천상의 신비주의를 테마로 한 고급스러운 타로 카드 웹사이트입니다.

## 🌟 주요 특징

### 디자인 컨셉
- **Aesthetic**: Celestial Mysticism (천상의 신비주의)
- **Color Palette**: 우주적 검정 배경 + 금색/은색 강조 + 보라/청록 액센트
- **Typography**: Cinzel (헤더) + Cormorant Garamond (본문)

### 핵심 기능

#### 1. 인터랙티브 타로 리딩
- 카테고리별 운세 (연애운, 사업운, 재물운, 건강운, 오늘의 운세)
- 12장의 카드 중 선택 가능
- 부드러운 카드 플립 애니메이션
- 상세한 카드 해석 제공

#### 2. 시각적 효과
- **별빛 배경**: 3개 레이어의 반짝이는 별 애니메이션
- **파티클 효과**: 주기적으로 나타나는 반짝임 효과
- **글로우 효과**: 금색 빛이 감도는 버튼과 요소들
- **패럴랙스 스크롤**: 스크롤에 따라 움직이는 요소들

#### 3. 반응형 디자인
- 데스크톱, 태블릿, 모바일 완벽 지원
- 유동적인 레이아웃
- 터치 인터랙션 최적화

## 🚀 사용 방법

1. **웹 브라우저에서 열기**
   - `index.html` 파일을 더블 클릭하거나
   - 웹 브라우저로 드래그 앤 드롭

2. **타로 리딩 시작**
   - 원하는 운세 카테고리 선택
   - 마음을 집중하고 카드 선택
   - 결과 확인 후 다시 보기 가능

## 🎨 디자인 하이라이트

### 색상 테마
```css
--color-cosmic-black: #0a0a1a    /* 우주 검정 */
--color-deep-space: #1a1a2e      /* 깊은 우주 */
--color-gold: #d4af37            /* 황금색 */
--color-silver: #c0c0c0          /* 은색 */
--color-mystic-purple: #8b5cf6   /* 신비한 보라 */
--color-celestial-cyan: #06b6d4  /* 천상의 청록 */
```

### 애니메이션
- **별빛 반짝임**: 10초 ~ 20초 주기
- **카드 플립**: 0.8초 3D 회전
- **페이드인**: 0.5초 ~ 1초
- **호버 효과**: 0.3초 부드러운 전환

### 특수 효과
- 마우스 움직임에 반응하는 별 레이어
- 스크롤 시 변화하는 네비게이션
- 주기적으로 생성되는 반짝임 파티클
- 회전하는 신비로운 원형 디자인
